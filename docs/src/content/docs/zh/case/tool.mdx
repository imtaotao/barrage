当我们停留在弹幕上时，可能会需要做一些操作，本章节会带你实现一个鼠标停留在弹幕上时弹出一个**工具栏**的用例，以**点赞/点踩**这个场景来演示。

:::note[提示]
本章节的组件以 **React** 来实现演示。
:::

## 1. 弹幕组件

```tsx title="Danmaku.tsx"
import { useState } from 'react';
import { Tool } from './Tool';

export function Danmaku({ danmaku }) {
  const content = danmaku.data.value;
  const [visible, setVisible] = useState(false);

  return (
    <div
      // 鼠标进入弹幕时，暂停弹幕的运动
      onMouseEnter={() => {
        danmaku.pause();
        setVisible(true);
      }}
      // 鼠标离开弹幕时，恢复运动
      onMouseLeave={() => {
        // 当处于冻结状态时，不要给恢复运动了（不过也视你的业务情况而定）
        if (manager.isFreeze()) return;
        danmaku.resume();
        setVisible(false);
      }}
    >
      {value}
      {visible ? <Tool /> : null}
    </div>
  );
}
```

## 2. 工具栏组件

```tsx title="Tool.tsx"
export function Tool() {
  // 发送点赞/点踩的请求，将结果存储在数据库
  const send = (type: string) => {
    fetch('http://xx', { method: 'POST', body: JSON.stringify({ type })});
  }

  return (
    <div>
      <botton onClick={() => send('good')}>点赞</button>
      <botton onClick={() => send('not-good')}>点踩</button>
    </div>
  );
}
```

## 3. 渲染弹幕

```tsx title="init.tsx"
import ReactDOM from 'react-dom/client';
import { create } from 'danmu';
import { Danmaku } from './Danmaku';

const manager = create({
  plugin: {
    // 将组件渲染到弹幕的内置节点上
    $createNode(danmaku) {
      ReactDOM.createRoot(danmaku.node).render(<Danmaku danmaku={danmaku} />);
    },
  },
});
```
