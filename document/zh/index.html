Danmu 💬

强大灵活且高度可扩展的弹幕库

Danmu 是一个用来管理，渲染，发送弹幕的库，覆盖了很多业务场景，并且提供了好用的扩展方式。
快速开始 API 参考 GitHub

多种渲染算法

默认提供了严格的碰撞检测算法，也提供了渐进式的碰撞检测算法，以及普通的全量实时渲染算法。

丰富的 API

提供了很多业务场景需要的 api，最大程度的简化开发方式。

样式高度可扩展

弹幕的渲染层由 CSS 提供，可以绘制任何适用 DOM 的弹幕需求，样式规则完全复用 CSS，没有额外的学习成本。

内核插件化

我们提供了丰富的钩子，这允许你方便的开发插件来满足更加定制化的需求，这是非常强大的能力。
